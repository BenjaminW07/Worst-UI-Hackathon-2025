## <u>Worst UI Hackathon Project — USF HackJam 2025</u>

2nd Place Winner — “Worst UI” Theme

This project was created during the USF HackJam 2025 hackathon, where one of the tracks was “Worst UI”. The goal was to intentionally design the most confusing, chaotic, and unusable interface possible. My teammate and I built a satirical version of the USF advising scheduler, exaggerating every frustrating UI pattern we could think of.
This project is intentionally bad that was the entire point of the competition and it earned 2nd place for creativity and execution.

### Purpose

This project is not meant to demonstrate clean UI/UX or production‑quality code. Instead, it highlights:
- Rapid prototyping under a 24‑hour deadline
- Creative interpretation of a theme
- Team collaboration
- Humor and design satire

### Tech Stack
- HTML
- CSS
- JavaScript
- Built quickly with the help of AI tools during the hackathon

### Running the Project

No setup required.
- Download or clone the repository
- Open the file named index.html in any web browser
- That’s it, the site runs locally with no dependencies
